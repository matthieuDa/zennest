# Domotique Location — Serrure Connectée, Thermostat... Indispensable ou Gadget ?

## La Révolution Silencieuse

Il y a 10 ans, la domotique était un luxe de riches : maisons intelligentes à 50 000 €, commandes vocales science-fiction, configurations complexes.

Aujourd'hui, **équiper un appartement locatif en domotique coûte 500-2 000 €** et se fait en 2h sans travaux.

Résultat : **20-30% des locations haut de gamme à Paris** intègrent au moins une serrure connectée ou un thermostat intelligent.

**Question** : Est-ce un **gadget** marketing ou un **investissement rentable** ?

Réponse : **Les deux**. Certains équipements sont indispensables (serrure, thermostat), d'autres sont superflus (volets roulants connectés, ampoules RGB).

## Les 5 Équipements Domotiques Rentables

### 1. La Serrure Connectée (Indispensable)

**Principe** : Remplace la serrure classique. Ouverture via **code numérique** (clavier), **smartphone** (Bluetooth), ou **badge RFID**.

**Modèles recommandés** :

- **Nuki Smart Lock** : 249 € (s'installe sur serrure existante, Bluetooth + WiFi)
- **Yale Linus** : 199 € (installation simple, batterie 1 an)
- **August Smart Lock** : 279 € (design premium, HomeKit compatible)

**Avantages** :

- ✅ **Zéro déplacement pour remises de clés** (économie 50-100 €/changement locataire)
- ✅ **Gestion Airbnb/bail mobilité** : codes temporaires automatiques
- ✅ **Sécurité** : traçabilité (qui entre/sort), verrouillage automatique
- ✅ **Modernité** : argument commercial fort

**Inconvénients** :

- Batterie à changer (1x/an, piles CR123)
- Connexion WiFi requise (box internet locataire OU pont WiFi propriétaire)
- Prix : 200-500 €

**ROI** : 2-3 ans (économie déplacements + différenciation).

**Notre avis** : **Indispensable** pour location courte durée (Airbnb, bail mobilité). **Très utile** pour location longue durée (confort + sécurité).

### 2. Le Thermostat Connecté (Très Rentable)

**Principe** : Remplace le thermostat classique. Pilotage à distance via smartphone, programmation horaire, détection présence.

**Modèles recommandés** :

- **Netatmo Smart Thermostat** : 179 € (design Starck, compatible toutes chaudières)
- **Google Nest Learning** : 249 € (apprentissage automatique, économies max)
- **Tado** : 199 € (multi-zones, géolocalisation)

**Avantages** :

- ✅ **Économies d'énergie** : 15-25% sur facture chauffage (programmation optimale)
- ✅ **Confort locataire** : pilotage depuis le canapé, température stable
- ✅ **Suivi conso** : détection surconsommations anormales (fenêtre ouverte, fuite)

**Inconvénients** :

- Installation électrique (remplacement thermostat existant, fil pilote 230V)
- WiFi requis

**ROI** : 1-2 ans (économies chauffage).

**Calcul** : Bien chauffé au gaz, facture 1 500 €/an.

- Économie avec thermostat : 20% = 300 €/an
- Coût thermostat : 180 €
- **ROI : 7 mois**

**Notre avis** : **Indispensable** si vous payez le chauffage (charges incluses). **Utile** même si locataire paie (confort = argument de visite).

### 3. Les Détecteurs de Fumée Connectés (Sécurité)

**Principe** : Détecteur de fumée + alerte smartphone (propriétaire averti immédiatement en cas d'incendie).

**Modèles recommandés** :

- **Nest Protect** : 119 € (détecte fumée + CO, auto-test mensuel)
- **Netatmo Smart Smoke Alarm** : 99 € (design, batterie 10 ans)

**Avantages** :

- ✅ **Alerte immédiate** : vous êtes prévenu même à distance
- ✅ **Sécurité renforcée** : détection CO (monoxyde de carbone)
- ✅ **Obligation légale respectée** (détecteur fumée obligatoire)

**Inconvénients** :

- Prix (vs 15 € détecteur classique)
- WiFi requis

**ROI** : Pas de ROI financier direct. **ROI sécurité** (évite sinistre 10 000-50 000 €).

**Notre avis** : **Recommandé** pour biens >150 000 €. Optionnel pour petits studios.

### 4. Les Détecteurs de Fuite d'Eau (Tranquillité)

**Principe** : Capteur posé au sol (sous évier, lave-linge, WC). Alerte smartphone si eau détectée.

**Modèles** :

- **Fibaro Flood Sensor** : 59 € (Z-Wave, batterie 2 ans)
- **Aqara Water Leak Sensor** : 29 € (Zigbee, économique)

**Avantages** :

- ✅ **Prévention dégâts des eaux** (alerte avant inondation massive)
- ✅ **Économie assurance** : sinistre évité = pas de franchise (500-1 000 €)

**Inconvénients** :

- Hub requis (Zigbee ou Z-Wave, coût 50-100 € supplémentaire)

**ROI** : Intangible. **Utile si** : lave-linge dans cuisine/sdb (risque fuite élevé).

**Notre avis** : **Optionnel** mais très rassurant pour propriétaires stressés.

### 5. Les Prises Connectées (Gadget Utile)

**Principe** : Prise avec interrupteur ON/OFF pilotable à distance. Permet d'éteindre un appareil oublié (fer à repasser, radiateur d'appoint).

**Modèles** :

- **TP-Link Tapo P100** : 9 € (WiFi, ultra-abordable)
- **Philips Hue Smart Plug** : 29 € (Zigbee, mesure conso)

**Avantages** :

- ✅ **Sécurité** : éteindre appareil oublié (fer, radiateur)
- ✅ **Économie** : couper veilles (TV, box)

**Inconvénients** :

- Utilité limitée (locataire gère lui-même)

**ROI** : Faible. **Gadget** sympathique mais pas essentiel.

**Notre avis** : **Optionnel**. Utile pour location courte durée (Airbnb) où vous gardez le contrôle.

## Les Équipements à Éviter (Surinvestissement)

### ❌ Volets Roulants Connectés

**Coût** : 300-600 €/volet.

**Utilité** : Quasi-nulle. Le locataire ouvre/ferme manuellement sans problème.

**ROI** : Jamais.

### ❌ Ampoules Connectées RGB

**Coût** : 50-100 €/ampoule.

**Utilité** : Gadget pur. Aucun locataire ne paie plus pour des ampoules qui changent de couleur.

**ROI** : Jamais.

### ❌ Caméras de Surveillance Intérieures

**Coût** : 100-300 €/caméra.

**Problème** : **Illégal** dans un logement loué (atteinte vie privée). Vous risquez poursuites pénales.

**Notre avis** : **Interdit**. Ne le faites jamais.

### ❌ Enceintes Connectées (Google Home, Alexa)

**Coût** : 50-150 €.

**Problème** : Le locataire apporte souvent la sienne. Celle fournie finit dans un placard.

**ROI** : Nul.

## L'Installation : DIY ou Pro ?

### DIY (Do It Yourself) — Économie 50%

**Équipements simples** :

- Serrure connectée : 30 min (notice claire, tournevis)
- Thermostat : 1h (raccordement fil pilote, attention 230V !)
- Détecteurs fumée : 5 min (vissage plafond)
- Prises : 0 min (brancher)

**Total temps** : 2h pour équiper un appartement complet.

**Économie** : 200-400 € (vs électricien/serrurier).

### Installation Pro — Sécurité + Garantie

**Coût** : 200-500 € (électricien + serrurier).

**Avantages** :

- Installation certifiée (assurance valide)
- Garantie travaux (2 ans)
- Pas de risque (court-circuit, mauvais branchement)

**Recommandation** : DIY si vous êtes bricoleur. **Pro si** thermostat multi-zones, alarme complexe, ou vous n'êtes pas à l'aise avec l'électricité (230V dangereux).

## Le ROI Global de la Domotique

### Simulation : T2 de 50m² à Paris (11e)

**Investissement domotique** : 800 € (serrure + thermostat + détecteurs).

**Gains annuels** :

2. **Économie déplacements remises de clés** : 2 changements locataires × 80 € = **160 €/an**

## Ce Que Personne Ne Vous Dit

Les serrures connectées **tombent en panne** après 3-5 ans (électronique défaillante, moteur grippé). Prévoir remplacement.

**Astuce** : Gardez toujours une **clé physique de secours** (en cas de panne électronique). Donnez-en une copie au locataire ET gardez-en une chez vous.

Autre point : certains locataires **refusent** la serrure connectée (peur piratage, surveillance). Proposez-leur le choix : code OU clé physique.

Enfin, les thermostats connectés nécessitent un **WiFi stable**. Si le locataire n'a pas de box internet (rare mais possible), le thermostat ne fonctionne plus en mode connecté (mais reste utilisable manuellement).

**Solution** : Installez un **mini-routeur 4G** (30 €/mois) si vous voulez garder le contrôle (utile pour Airbnb, inutile pour location longue).
